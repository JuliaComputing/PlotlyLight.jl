Introduction

PlotlyLight.jl is a speedy lightweight interface for creating Plotly.js plots from Julia.

‚ú® Features

  • üöÄ Fastest time-to-first-plot in Julia!
  • üåê Use the Plotly.js Javascript documentation directly. No magic syntax: Just JSON3.write.
  • üìÇ Set deeply-nested keys easily, e.g. myplot.layout.xaxis.title.font.family = "Arial".
  • üìä The Same built-in themes as Plotly's python package.

üöÄ Quickstart

using PlotlyLight

# Change template
preset.template.plotly_dark!()

# Make Plot
p = plot(x = 1:20, y = cumsum(randn(20)), type="scatter", mode="lines+markers")

# Make changes
p.layout.title.text = "My Title!"

# Re-display to see updated plot
p
